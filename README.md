# Voxel Engine

### Features

This repo will be used to build a modular voxel rendering tool that will offer:
  - Occlusion culling
  - Collision dectection
  - Dynamic lighting
  - Particle physics simulation
  - Procedural terrain noise generation
  - Perspective camera
